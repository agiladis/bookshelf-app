# Bookshlef App

Simple Bookshelf Web Service

## Description

Bookshlef app is simple web service develop with nodeJS environment It can handle POST, GET, PUT, DELETE request

## Getting Started

### Dependencies

* hapi
* nanoid

### Executing program

* Commit this repository
* install dependecies
```
npm install
```
* run web service in local
```
npm run start-dev
```
